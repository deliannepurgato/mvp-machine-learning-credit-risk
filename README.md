# MVP - Previsão de Risco de Inadimplência

**Disciplina:** Machine Learning e Analytics

## Objetivo

Desenvolver e avaliar modelos de Machine Learning para prever o risco de inadimplência de clientes com base em características financeiras, pessoais e histórico de crédito.

## Dataset

O conjunto de dados utilizado contém informações relacionadas a empréstimos, perfil financeiro e histórico de crédito dos clientes.

* Registros: 32.581
* Atributos: 12
* Tipo de problema: Classificação supervisionada

## Modelos Avaliados

* DummyClassifier (Baseline)
* Logistic Regression
* Decision Tree
* K-Nearest Neighbors (KNN)
* Random Forest

## Melhor Modelo

Random Forest otimizado utilizando RandomizedSearchCV.
